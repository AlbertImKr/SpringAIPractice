Spring AI 소개

Spring AI는 AI 엔지니어링을 위한 애플리케이션 프레임워크입니다.
이 프레임워크는 Spring 생태계의 이동성과 모듈형 설계 원칙을 AI 도메인에 적용합니다.

주요 기능:
1. 다양한 AI 모델과의 통합
2. 채팅, 텍스트-이미지, 임베딩 모델 지원
3. 벡터 데이터베이스 지원
4. 출력 파싱 기능
5. ETL 프레임워크

Spring AI는 다음과 같은 AI 제공업체를 지원합니다:
- OpenAI
- Azure OpenAI
- Amazon Bedrock
- Anthropic Claude
- Google VertexAI
- Hugging Face
- Ollama

Spring AI의 핵심 개념:
- ChatModel: 대화형 AI 모델
- EmbeddingModel: 텍스트 임베딩 생성
- VectorStore: 벡터 데이터 저장
- ImageModel: 이미지 생성 및 분석
- AudioModel: 음성 처리
